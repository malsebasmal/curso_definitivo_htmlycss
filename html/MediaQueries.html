Que son los MediaQueries?

Las media queries son una característica de CSS que permite aplicar estilos diferentes según ciertas características del dispositivo o de la pantalla. Las media queries se utilizan junto con los breakpoints para adaptar el diseño en diferentes situaciones. Por ejemplo:

@media (max-width: 768px) { Estilos aplicados cuando el ancho de la pantalla es menor o igual a 768px }
