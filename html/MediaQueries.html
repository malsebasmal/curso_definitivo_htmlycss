Que son los MediaQueries?

Las media queries son una característica/declaración de CSS que permite definir mediante los breakpoints en qué punto/pixeles de la pantalla, las propiedades/características/elementos de esta, van a cambiar. Por ejemplo:
-De pasar de 4 columnas, a 2 columnas
-En que punto de la resolución/pixeles se aplica el menú mobile.

Normalmente una página web tiene muchos @media (MediaQueries), donde cada una de estas MediaQueries tendrán nuestras medidas especificas con las cuales aplicaremos el código responsive.

Entendido esto, tendremos que entender estos conceptos:
-Sintaxis de un MediaQueries

@media (max-width: 768px) { Estilos aplicados cuando el ancho de la pantalla es menor o igual a 768px }
